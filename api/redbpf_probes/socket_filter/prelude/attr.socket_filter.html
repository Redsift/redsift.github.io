#[socket_filter]

Attribute macro that must be used to define socket filter probes.

See also the socket filter API provided by redbpf-probes.

Example

use redbpf_probes::socket_filter::prelude::*;

#[socket_filter]
fn probe(skb: SkBuff) -> SkBuffResult {
    Ok(SkBuffAction::SendToUserspace)
}
