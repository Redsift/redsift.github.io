@mm_users: The number of users including userspace.

Use mmget()/mmget_not_zero()/mmput() to modify. When this drops to 0 (i.e. when the task exits and there are no other temporary reference holders), we also release a reference on @mm_count (which may then free the &struct mm_struct if @mm_count also drops to 0).

mm_count: atomic_t

@mm_count: The number of references to &struct mm_struct (@mm_users count as 1).

Use mmgrab()/mmdrop() to modify. When this drops to 0, the &struct mm_struct is freed.
